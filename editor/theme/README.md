# Color Theme Reference

| Color Type | Description |
| :--- | :--- |
| `background` | Background for widgets, dark icon |
| `background_light` | Background for things like cards and tree widget headers |
| `background_active` | Background for things like button hovers |
| `foreground` | Color for light icon, text etc |
| `foreground_dark` | Color for secondary text, placeholder text etc |
| `foreground_disabled` | Color for disabled text, icon |
| `foreground_active` | Color for active foreground buttons, slider buttons etc |
| `border` | Border color |
